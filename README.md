Weather App

Android приложение для отображения погоды с использованием внешнего API.

Stack: Kotlin, Jetpack Compose, MVVM, Volley, Coroutines  

Архитектура:
UI → ViewModel → API  

- Получение данных с Weather API  
- Парсинг JSON  
- Отображение прогноза (часы / дни)  
- Управление состоянием через Compose
